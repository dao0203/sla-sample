# sla-sample
学習支援コーナー内のGithub講座のサンプルリポジトリ

## 環境構築する方法
### 前提条件
以下のソフトウェアがインストールされていることを前提条件とします。

- Git
- VSCode
- Docker Desktop（Linuxの場合はdocker-engine + docker-compose）

### Mac/Linuxの場合
以下のコマンドを実行するだけで仮想環境の構築〜実行を行ってくれます。
```
./start.sh
```

### Windowsの場合
Windowsの場合も以下のコマンドを実行するだけで仮想環境の構築〜実行を行ってくれます。
```
./start.bat
```
